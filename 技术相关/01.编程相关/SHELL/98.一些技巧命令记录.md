##### 利用lsof查看某类进程或某个进程或某类文件的读写量
```
awk '
/mysqld.*tmp/{
  total += $7;
}
/^Sun Mar 28/ && total {
  printf "%s %7.2f MB\n", $4, total/1024/1024;
  total = 0;
}' lsof.txt
```
##### 星期大小写转换

```
week_num=`date +%w`
week_array=(日 一 二 三 四 五 六)
WEEK=${week_array[$week_num]}
```
