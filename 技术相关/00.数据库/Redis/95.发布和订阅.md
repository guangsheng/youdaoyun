- 利用频道（channel）实现发布订阅
- 服务之间的消息解耦、聊天室


#### 命令
```
发布消息： publish channel message
订阅消息： subscribe channel [channel ...]
取消订阅： unsubscribe [channel [channel ...]]
按照模式订阅和取消订阅: psubscribe/punsubscribe [pattern [pattern ...]]

查看至少有一个订阅者的频道： pubsub channels [pattern]
查看频道订阅数： pubsub numsub [channel ...]
查看模式订阅数： pubsub numpat
```


#### 样例
```
##发布
127.0.0.1:6379> publish channel:sport basketball
(integer) 0
127.0.0.1:6379> publish channel:music "xuwei"
(integer) 0
127.0.0.1:6379> publish channel:sport "basketball game"
(integer) 1
127.0.0.1:6379>

##订阅
127.0.0.1:6379> SUBSCRIBE channel:sport
Reading messages... (press Ctrl-C to quit)
1) "subscribe"
2) "channel:sport"
3) (integer) 1
1) "message"
2) "channel:sport"
3) "basketball game"

##查看频道情况
127.0.0.1:6379> pubsub channels
1) "channel:sport"
127.0.0.1:6379> pubsub numsub
(empty list or set)
```
