- 一种高效的计算总数的方法
- 并不是一种数据结构（实际类型是字符串类型），而是一种基数算法
- 占用内存**非常小**，但是存在错误率

#### 适合场景
- 只是为了计算总数，不需要获取单条数据
- 可以容忍一定的误差率

#### 命令
```
pfadd key element [element ...]
pfcount key [key ...]
pfmerge destkey sourcekey [sourcekey ...]
```
